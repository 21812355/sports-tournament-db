## Project Description
A full-stack web application to manage sports tournaments including
teams, players, matches, and statistics. Built with Python/Streamlit
frontend and PostgreSQL (Supabase) cloud database.

## Technology Stack
- **Frontend:** Python / Streamlit
- **Database:** PostgreSQL on Supabase
- **Deployment:** Streamlit Cloud

## Live App
[Click here to open the app](https://sports-tournament-db-xudthg3vzmpwavxvx4rw7u.streamlit.app/)

## Database
- 7 tables with PKs, FKs, constraints
- 7 SQL queries (joins, subqueries, aggregations)
- 5 PL/SQL blocks (procedures, functions, triggers)
